# Лабораторная работа "Создание Wireframe для своей предметной области"

Используя материалы лекции ["Прототипы экранов и окон пользовательского интерфейса"](./wireframe.md) разработать прототипы экранов для своей предметной области:

* **админка** - десктопное приложение для заполнения справочников (аналог "продукции" и "материалов" в лекциях)
* **клиентское приложение** - мобильное и/или веб- приложение для онлайн оформления заказа в вашем магазине/сервисе

В репозитории должны быть скриншоты разработанных интерфейсов и ссылка на интерфейс, если использовали веб-приложения (типа figma)
